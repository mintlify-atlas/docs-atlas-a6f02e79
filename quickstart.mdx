---
title: "Quickstart: run your Solana dApp in five minutes"
description: "Clone the PubKey UI Starter repository, install dependencies, and run both the web app and API server locally in under five minutes."
---

PubKey UI Starter gets you from zero to a running Solana dApp in a few commands. This guide walks through cloning the repository, installing dependencies, starting both servers, and connecting a wallet.

<Steps>
  <Step title="Prerequisites">
    Make sure you have the following installed before you begin:

    - **Node.js** v20 or higher
    - **PNPM** v8 or higher
    - **Git**

    <Tip>
      If you don't have PNPM installed, use Node.js's built-in Corepack:

      ```bash
      corepack enable && corepack prepare pnpm@8 --activate
      ```
    </Tip>
  </Step>

  <Step title="Clone and install">
    Clone the repository into a directory of your choice and install dependencies:

    <CodeGroup>

    ```bash Clone the repository
    git clone https://github.com/pubkeyapp/pubkey-ui-starter my-app
    cd my-app
    ```

    ```bash Install dependencies
    pnpm install
    ```

    </CodeGroup>

    PNPM installs all workspace dependencies for both the `web` and `api` projects from the root `package.json`.
  </Step>

  <Step title="Start the API server">
    In one terminal, start the Express API:

    ```bash
    pnpm dev:api
    ```

    The API server listens on port `3000` by default and mounts all routes under `/api`. You can override the port with a `PORT` environment variable.
  </Step>

  <Step title="Start the web app">
    In a second terminal, start the React frontend:

    ```bash
    pnpm dev:web
    ```

    This starts the Nx webpack dev server with hot module replacement enabled. The app proxies `/api` requests to the API server running on port 3000, so both servers must be running at the same time.
  </Step>

  <Step title="Connect your wallet">
    Open your browser and navigate to the address shown by the webpack dev server (typically `http://localhost:4200`).

    1. Click the wallet button in the top-right corner.
    2. Connect Phantom, Solflare, or any Wallet Standard-compatible wallet.
    3. Use the cluster dropdown to switch to **devnet**.
    4. Navigate to the **Account** page to see your SOL balance and token accounts.

    <Note>
      If you don't have a wallet extension installed, Phantom and Solflare both offer browser extensions for Chrome, Firefox, and Brave.
    </Note>
  </Step>
</Steps>
