## Get started

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/pubkeyapp/pubkey-ui-starter my-app
    cd my-app
    pnpm install
    ```
  </Step>
  <Step title="Start the development servers">
    Run the API and web app concurrently in separate terminals:
    ```bash
    pnpm dev:api   # starts Express API on port 3000
    pnpm dev:web   # starts React app with webpack dev server
    ```
  </Step>
  <Step title="Connect a wallet">
    Open your browser, connect a Solana wallet (e.g. Phantom or Solflare), and select **devnet** from the cluster dropdown to start exploring your account.
  </Step>
  <Step title="Build your dApp">
    Add your own features to `web/src/app/features/`, extend the API in `api/src/lib/features/`, or scaffold an Anchor program with `create-solana-dapp`.
  </Step>
</Steps>

<Note>
  PubKey UI is under active development. Minor and major API changes can occur at any time. Pin your dependencies and review the changelog when upgrading.
</Note>
