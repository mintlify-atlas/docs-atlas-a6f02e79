The PubKey UI Starter web app is a React 18 single-page application that combines Mantine UI v7 for components and theming, TanStack Query v5 for server-state management, Jotai v2 for local atom-based state, and React Router v6 for client-side navigation. It is bundled with Webpack inside an Nx monorepo and connects to Solana through the Wallet Adapter libraries.

## Technology stack

  <Card title="React 18 + React DOM" icon="react">

  </Card>
  <Card title="Jotai v2" icon="atom">

  </Card>
</CardGroup>

## Application providers

All providers are composed in `web/src/app/app.tsx`. The nesting order matters: each provider can only consume context from providers above it in the tree.

```tsx app.tsx
import { UiThemeProvider } from '@pubkey-ui/core'
import '@pubkey-ui/core/index.esm.css'

import { QueryClient, QueryClientProvider } from '@tanstack/react-query'
import { AppRoutes, ThemeLink } from './app-routes'
import { ClusterProvider } from './features/cluster/cluster-data-access'
import { SolanaProvider } from './features/solana/solana-provider'

const client = new QueryClient()

export function App() {
  return (
    <QueryClientProvider client={client}>
      <UiThemeProvider link={ThemeLink}>
        <ClusterProvider>
          <SolanaProvider>
            <AppRoutes />
          </SolanaProvider>
        </ClusterProvider>
      </UiThemeProvider>
    </QueryClientProvider>
  )
}
```

The provider stack from outermost to innermost:

| Provider | Package | Role |
|---|---|---|
| `QueryClientProvider` | `@tanstack/react-query` | Supplies the shared `QueryClient` instance to all hooks |
| `UiThemeProvider` | `@pubkey-ui/core` | Wraps Mantine's `MantineProvider`; receives `ThemeLink` to integrate React Router with Mantine's link components |
| `ClusterProvider` | local | Stores the active Solana cluster (Mainnet, Devnet, Localnet) in Jotai atoms |
| `SolanaProvider` | local | Wires up `ConnectionProvider`, `WalletProvider`, and `WalletModalProvider` using the active cluster endpoint |
| `AppRoutes` | local | Renders the React Router route tree inside `AppLayout` |
